# Prompt 日志说明

本文件夹存放 P1 阶段需求分析过程中与 AI 的主要交互记录，作为《AI 协作反思日志 #1》的支撑材料提交。

## 文件说明

### 1. ai_interaction_log.md
主要用于需求发散与前期需求启发，从“如果你是用户，你觉得会有哪些需求”的角度整理校园互助平台中需求方、服务方和普通用户的真实关注点，可用于支持问卷设计和需求初步整理。

### 2. aistudio交互记录1.md
主要用于需求规格说明书初稿生成，以及任务 3.2 所要求的用户故事生成。

### 3. aistudio交互记录2.md
主要用于需求规格说明书初稿生成，以及结合问卷结果、访谈笔记生成任务 3.1 所需的系统用例图和核心详细用例描述。

### 4. chatgpt交互记录_P1补充.md
主要用于补充说明 P1 阶段整体任务、任务 3（用例建模与用户故事）的具体要求，以及 Prompt 日志本身应如何理解和整理。该文件选取了与 P1 直接相关的 3 组高质量问答，作为补充性交互记录提交。

## 说明

1. 以上文件均为 P1 阶段真实 AI 交互记录或基于真实交互整理而成的记录文件。
2. 其中前三份为原始或近原始交互记录，第四份为基于实际对话内容筛选整理的补充记录。
3. 这些记录共同对应了 P1 阶段中的以下任务：
   - 需求发散与调研启发
   - 需求规格说明书生成
   - 用例图与详细用例生成
   - 用户故事生成
   - Prompt 日志整理与反思支撑
4. 本组提交 Prompt 日志时，保留原始交互内容，同时增加本说明文件，以便更清楚地展示每份记录与阶段任务的对应关系。
