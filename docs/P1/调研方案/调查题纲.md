# 用户访谈提纲

## 访谈主题
校园互助服务平台用户需求访谈

## 访谈目的
了解大学生在校园生活中的真实互助需求、当前寻求帮助的方式，以及对校园互助平台功能、信用机制和使用体验的期待，为后续需求分析提供依据。

## 访谈问题

1. 你在校园生活中，最近遇到过哪些需要别人帮忙的事情？

2. 当你需要帮助时，通常会通过什么方式找人？

3. 如果有一个专门的校园互助平台，你最希望它优先提供哪些功能？

4. 如果平台提供实名认证、信用分、历史评价，你觉得哪个最有用？为什么？

5. 如果你用过这个平台一次后，什么样的体验会让你愿意继续使用，甚至推荐给别人？

## 说明
- 本访谈采用开放式问题形式，鼓励受访者结合个人真实经历自由回答。
- 访谈结果将用于校园互助服务平台的需求分析与功能设计。
